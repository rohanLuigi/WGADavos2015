Accuracy of GBLUP¶

Theory¶

Best Linear Prediction (BLP)¶

Assume the breeding value $u_i$ can be modeled as \begin{equation} u_i = \mathbf{q}_i'\mathbf{\alpha}, \end{equation} where $\mathbf{q}_i$ are the unobserved QTL genotypes. For simplicity all variables will be assumed to have mean zero.

Let $\mathbf{x}_i$ denote a vector of marker genotypes. Then, BLP of $u_i$ given $\mathbf{x}_i$ is \begin{align} \hat{u}_i &= Cov(u_i,\mathbf{x}_i')[Var(\mathbf{x}_i)]^{-1}\mathbf{x}_i \\ &= \mathbf{\alpha}'\mathbf{V}_{QX}\mathbf{V}^{-1}_{XX}\mathbf{x}_i\\ &= \mathbf{\beta}'\mathbf{x}_i, \end{align} where, $\mathbf{\beta}' = \mathbf{\alpha}'\mathbf{V}_{QX}\mathbf{V}^{-1}_{XX}$, and for example, $V_{QX} = Cov(\mathbf{q}_i,\mathbf{x}_i')$.

Given $\mathbf{\beta}$ is known, the squared correlation between the predictor and the predictand is \begin{align} r^2(\hat{u}_i,u_i) &= \frac{\mathbf{\alpha}'\mathbf{V}_{QX}\mathbf{V}^{-1}_{XX}Cov(\mathbf{x}_i,u_i)} {\mathbf{\alpha}'\mathbf{V}_{QQ}\mathbf{\alpha}} \\ &=\frac{\mathbf{\alpha}'\mathbf{V}_{QX}\mathbf{V}^{-1}_{XX}\mathbf{V}_{XQ}\mathbf{\alpha}} {\mathbf{\alpha}'\mathbf{V}_{QQ}\mathbf{\alpha}}\\ &= \frac{\mathbf{\beta}'\mathbf{V}_{XX}\mathbf{\beta}} {\mathbf{\alpha}'\mathbf{V}_{QQ}\mathbf{\alpha}}\\ &= q^2 \end{align} In practice, $\mathbf{\beta}$ has to be estimated. But, because the QTL genotypes, $\mathbf{q}_i$, are not observed, $V_{QX}$ is estimated indirectly.

Ridge Regression¶

Given the trait phenotype has model \begin{equation} y_i = \mathbf{q}_i\mathbf{\alpha} + e_i, \end{equation} $Cov(u_i,\mathbf{x}_i')=Cov(y_i,\mathbf{x}_i')$. So, if a vector $\mathbf{y}$ of trait phenotypes and a matrix $\mathbf{X}$ of marker genotypes were available, $\mathbf{C} = Cov(u_i,\mathbf{x}_i')$ can be estimated as $\hat{\mathbf{C}}=\mathbf{y}'\mathbf{X}/n$, and $\mathbf{V}_{XX}$ can be estimated as $\hat{\mathbf{V}}_{XX}=\mathbf{X}'\mathbf{X}/n$, provided the number $n$ of observations is sufficiently large.

Currently, $n$ is lower than the number $p$ of markers. So, $\mathbf{V}_{XX}$ estimated as above would be singular, but $\mathbf{\beta}$ can be estimated by ridge regression as \begin{equation} \hat{\mathbf{\beta}} = [\mathbf{X}'\mathbf{X} + \mathbf{I}\lambda]^{-1}\mathbf{X}'\mathbf{y}. \end{equation} Next, we will derive the squared correlation between $u_i$ and $\tilde{u}_i=\mathbf{x}_i'\hat{\mathbf{\beta}}$.

Derivation of $r^2(u_i,\tilde{u}_i)$¶

Although we used ridge regression to estimate $\mathbf{\beta}$, we will assume $\mathbf{\alpha}$ and $\mathbf{\beta}$ are fixed effects, which is typical in quantitative genetics. Further, we will assume that the candidate and the training individuals are randomly sampled from some reference population. The covariance between $u_i$ and $\tilde{u}_i$ is \begin{align} Cov(u_i,\tilde{u}_i) &= Cov(\mathbf{q}_i'\mathbf{\alpha},\mathbf{x}_i'\hat{\mathbf{\beta}})\\ &= E [Cov(\mathbf{q}_i'\mathbf{\alpha},\mathbf{x}_i'\hat{\mathbf{\beta}}|\mathbf{q}_i,\mathbf{x}_i )] + Cov[E(\mathbf{q}_i'\mathbf{\alpha}|\mathbf{q}_i,\mathbf{x}_i),E(\mathbf{x}_i'\hat{\mathbf{\beta}}|\mathbf{q}_i,\mathbf{x}_i)]\\ &= 0 + Cov(\mathbf{q}_i'\mathbf{\alpha}, \mathbf{x}_i'E(\hat{\mathbf{\beta}})) \\ &= Cov(\mathbf{q}_i'\mathbf{\alpha}, \mathbf{x}_i'\mathbf{\beta}), \end{align}

where $Cov[E(\mathbf{q}_i'\mathbf{\alpha}|\mathbf{q}_i,\mathbf{x}_i),E(\mathbf{x}_i'\hat{\mathbf{\beta}}|\mathbf{q}_i,\mathbf{x}_i)]=0$ because conditional on $\mathbf{q}_i$, $\mathbf{q}_i'\mathbf{\alpha}$ is not random. On the last line, we have treated $\hat{\mathbf{\beta}}$ as unbiased. This is not true of the ridge-regression estimator, but the bias will become negligible as number $n$ of observations in training increases. This expression is an upper bound that will be reached only as $n$ goes to infinity. So, \begin{align} Cov(u_i,\tilde{u}_i) &= Cov(\mathbf{q}_i'\mathbf{\alpha}, \mathbf{x}_i'\mathbf{\beta}) \\ &= \mathbf{\alpha}'\mathbf{V}_{QX}\mathbf{\beta}\\ &= \mathbf{\alpha}'\mathbf{V}_{QX}\mathbf{V}^{-1}_{XX}\mathbf{V}_{XQ}\mathbf{\alpha}. \end{align}
It may be useful to see that this can also be expressed as

\begin{align} Cov(u_i,\tilde{u}_i) &= \mathbf{\alpha}'\mathbf{V}_{QX}\mathbf{\beta}\\ &= \mathbf{\beta}'\mathbf{V}_{XX}\mathbf{\beta}. \end{align}

Next, the variance of our predictor is \begin{align} Var(\mathbf{x}_i'\hat{\mathbf{\beta}}) &= E[Var(\mathbf{x}_i'\hat{\mathbf{\beta}}|\mathbf{x}_i)] + Var[E(\mathbf{x}_i'\hat{\mathbf{\beta}}|\mathbf{x}_i)]\\ &= E[\mathbf{x}_i'Var(\hat{\mathbf{\beta}})\mathbf{x}_i] + \mathbf{\beta}'\mathbf{V}_{XX}\mathbf{\beta}\\ &= \text{trace}[Var(\hat{\mathbf{\beta}})V_{XX}] + \mathbf{\alpha}'\mathbf{V}_{QX}\mathbf{V}^{-1}_{XX}\mathbf{V}_{XQ}\mathbf{\alpha}, \end{align} and the variance of the predictand is $\mathbf{\alpha}'\mathbf{V}_{QQ}\mathbf{\alpha}$. In this derivation also we have assumed the ridge-regression estimator is unbiased, which is only approximately true. The second term in the above expression is an upper bound, which is reached only when the regression coefficients are not shrunk as n tends to infinity. So, both the numerator and denominator are biased up.

Combining these results gives: \begin{equation} r^2(u_i,\tilde{u}_i) = \frac{(\mathbf{\beta}'\mathbf{V}_{XX}\mathbf{\beta})^2} {\text{trace}[Var(\hat{\mathbf{\beta}})V_{XX}]\mathbf{\alpha}'\mathbf{V}_{QQ}\mathbf{\alpha}}. \end{equation}

To proceed, we need an expression for $Var(\hat{\mathbf{\beta}})$. The variance of the ridge-regression estimator can be written as \begin{align} Var(\hat{\mathbf{\beta}}) &= E\{Var[(\mathbf{X}'\mathbf{X} + \mathbf{I}\lambda)^{-1}\mathbf{X}'\mathbf{y})]|\mathbf{X}\} + Var[E(\hat{\mathbf{\beta}}|\mathbf{X})] \end{align} Again, if we treat the ridge-regression estimator as unbiased, the second term in the above expression becomes null, and \begin{align} Var(\hat{\mathbf{\beta}}) &= E\{Var[(\mathbf{X}'\mathbf{X} + \mathbf{I}\lambda)^{-1}\mathbf{X}'\mathbf{y})]|\mathbf{X}\}\\ &= E [\mathbf{C}^{-1}\mathbf{X}'Var(\mathbf{y})\mathbf{X}\mathbf{C}^{-1}], \end{align} where $\mathbf{C} = (\mathbf{X}'\mathbf{X} + \mathbf{I}\lambda)$. Recall that we assume that the training individuals are randomly sampled from a reference population. Thus, $Var(\mathbf{y})$ is a diagonal matrix with diagonal element $j$ equal to \begin{align} Var(y_j|\mathbf{x}_j) &= Var(u_j|\mathbf{x}_j) + e_j\\ &= Var(\mathbf{q}_j\mathbf{\alpha}|\mathbf{x}_j) + e_j\\ &\approx \mathbf{\alpha}'\mathbf{V}_{QQ}\mathbf{\alpha} - \mathbf{\beta}'\mathbf{V}_{XX}\mathbf{\beta} + Var(e_j), \end{align} where we have approximated the conditional variance by assuming multivariate normality for $\mathbf{q}_j$ and $\mathbf{x}_j$ So, $Var(\hat{\mathbf{\beta}})$ becomes \begin{align} Var(\hat{\mathbf{\beta}}) &\approx E[\mathbf{C}^{-1}(\mathbf{X}\mathbf{X})\mathbf{C}^{-1}](\mathbf{\alpha}'\mathbf{V}_{QQ}\mathbf{\alpha} - \mathbf{\beta}'\mathbf{V}_{XX}\mathbf{\beta} + Var(e_j)) \\ &\approx [n\mathbf{V}_{XX} + \mathbf{I}\lambda]^{-1}(n\mathbf{V}_{XX})[n\mathbf{V}_{XX} + \mathbf{I}\lambda]^{-1} (\mathbf{\alpha}'\mathbf{V}_{QQ}\mathbf{\alpha} - \mathbf{\beta}'\mathbf{V}_{XX}\mathbf{\beta} + Var(e_j)), \end{align} where we have approximated the expectation on the first line as: \begin{equation} E[\mathbf{C}^{-1}(\mathbf{X}\mathbf{X})\mathbf{C}^{-1}] \approx [n\mathbf{V}_{XX} + \mathbf{I}\lambda]^{-1}(n\mathbf{V}_{XX})[n\mathbf{V}_{XX} + \mathbf{I}\lambda]^{-1}. \end{equation}
This approximation is not good when $n$ is less than the number $p$ of markers.

Finally, $r^2(u_i,\tilde{u}_i)$ becomes \begin{align} r^2(u_i,\tilde{u}_i) &= \frac{(\mathbf{\beta}'\mathbf{V}_{XX}\mathbf{\beta})^2} {[d(\mathbf{\alpha}'\mathbf{V}_{QQ}\mathbf{\alpha} - \mathbf{\beta}'\mathbf{V}_{XX}\mathbf{\beta} + Var(e_j))+ \mathbf{\beta}'\mathbf{V}_{XX}\mathbf{\beta}] [\mathbf{\alpha}'\mathbf{V}_{QQ}\mathbf{\alpha}]}\\ &= \frac{[Var(\hat{u}_i)]^2}{[Var(\tilde{u}_i)][Var(u_i)]} \end{align} where $d=\text{trace}[(n\mathbf{V}_{XX} + \mathbf{I}\lambda)^{-1}(n\mathbf{V}_{XX})(n\mathbf{V}_{XX} + \mathbf{I}\lambda)^{-1}\mathbf{V}_{XX}]$.

Dividing the numerator and denominator by $[Var(u_i)]^2$ gives \begin{equation} r^2(u_i,\tilde{u}_i) = \frac{q^4} {d(1 - q^2 + \sigma^2_e)+q^2}, \end{equation} where $q^2 = \frac{\mathbf{\beta}'\mathbf{V}_{XX}\mathbf{\beta} }{\mathbf{\alpha}'\mathbf{V}_{QQ}\mathbf{\alpha} }$, which is the proportion of the genetic variance explained by the markers, and $\sigma^2_e = \frac{Var(e_j)}{\mathbf{\alpha}'\mathbf{V}_{QQ}\mathbf{\alpha}}$, which is the environmental variance when the trait is scaled to have a genetic variance of 1.0.
Note that $q^2$ does not depend on the number $n$ of training individuals, but $d$ is inversely related to $n$. So, the upper bound for $r^2(u_i,\tilde{u}_i)$ is $q^2$.

A Better Approximation of $Var(\hat{\mathbf{\beta}})$¶

Suppose the trait is scaled to have a genetic variance of one. Then, variance of the least-squares estimator of $\mathbf{\beta}$ is \begin{equation} Var(\hat{\mathbf{\beta}}) = (\mathbf{X}'\mathbf{X})^{-1}(1−q^2+σ^2_e), \end{equation} where if $n>p$ and normality is used to approximate the distribution of marker covariates, $(\mathbf{X}'\mathbf{X})^{-1}$ has an inverse Wishart distribution with scale matrix $\mathbf{V}_{XX}$ and $n$ degrees of freedom. Then, the expected value of $(\mathbf{X}'\mathbf{X})^{-1}$ is \begin{equation} E[(\mathbf{X}'\mathbf{X})^{-1}] = \frac{\mathbf{V}_{XX}^{-1}}{n-p-1} \end{equation}

Using this result, \begin{align} E[\mathbf{x}_i'Var(\hat{\mathbf{\beta}})\mathbf{x}_i] &= \text{trace}(\mathbf{V}^{-1}_{XX}\mathbf{V}_{XX})\frac{(1−q2+σ^2_e)}{n-p-1}\\ &= \frac{p(1−q2+σ^2_e)}{n-p-1}. \end{align} In least-squares, $p$ is equal to the degrees of freedom of the model. In ridge-regression, the degrees of freedom of the model are computed as $\text{tr}[\mathbf{X}(\mathbf{X}'\mathbf{X}+\mathbf{I}\lambda)^{-1}\mathbf{X}']$, and so $E[\mathbf{x}_i'Var(\hat{\mathbf{\beta}})\mathbf{x}_i]$ is approximated as \begin{equation} E[\mathbf{x}_i'Var(\hat{\mathbf{\beta}})\mathbf{x}_i] = \frac{\nu_M(1−q2+σ^2_e)}{n-\nu_M-1}, \end{equation}
where $\nu_M=\text{tr}[(n\mathbf{V}_{XX}+\mathbf{I}\hat{\lambda})^{-1}n\mathbf{V}_{XX}]$ and $\hat{\lambda}=\text{tr}(\mathbf{V}_{XX})$. Because $\mathbf{V}_{XX}$ has rank $p$, unless $\hat{\lambda}$ is large enough, $\nu_M$ can be larger than $n$. I have verified that with $\hat{\lambda}=\text{tr}(\mathbf{V}_{XX})$, $\nu_M$ is less than $n$ even for $n=1$.

Then, \begin{align} r^2(u_i,\tilde{u}_i) &= \frac{q^4} {\frac{\nu_M(1−q2+σ^2_e)}{n-\nu_M-1}+q^2}\\ &= \frac{q^4(n-\nu_M-1)} {\nu_M(1−q2+σ^2_e) + (n-\nu_M-1)q^2}\\ &= \frac{q^2(n-\nu_M-1)} {\nu_M\theta + (n-\nu_M-1)}, \end{align} where $\theta = \frac{1−q2+σ^2_e}{q^2}$.

Numerical Examples¶

In the following numerical examples, the covariances between genotype covariates $w_k$ and $w_l$ are computed as \begin{equation} Cov(w_k,w_l) = 0.5[\frac{1}{4N_ec + 1}]^{-1/2}, \end{equation} where $c$ is the recombination rate between the loci, and we have assumed $Var(w_k)=0.5$ for all loci. Further, the heritability of the trait has been set to 0.5. Then, for a trait that is scaled to have a genetic variance of 1.0, $\lambda$ was set to $p/2$, where $p$ is the number of markers. Added least-squares estimator of $\mathbf{\beta}$. For the least-squares estimator, $d=\frac{p}{n-p-1}$.

class GBLUP:
    def __init__(self,L,Ne,p):
        self.p = p/nChr
        x = np.arange(0.0,self.p,1.0)
        C = np.mat(np.zeros((self.p,self.p)))
        for i in x:
            for j in x:
                C[i,j] = abs(i-j)/self.p * L
        R = 0.5*(1 - exp(-2*C))
        self.V = kron(identity(nChr),np.mat(0.5*sqrt(1/(4.0*Ne*R + 1))))
        #print self.V[0:3,0:3]
    def getV(self):
        return self.V
    def q2(self,QTL):
        MRK=delete(range(p),QTL)
        self.VQX=self.V[QTL,:][:,MRK]
        self.VXX=self.V[MRK,:][:,MRK]
        self.VQQ=self.V[QTL,:][:,QTL]
        self.p = shape(self.VXX)[0]
        alpha = matrix(repeat(1,nQTL))
        a = alpha*self.VQX*self.VXX.I*self.VQX.T*alpha.T
        b = alpha*self.VQQ*alpha.T
        return (a/b)[0,0]
    def r2(self,n,QTL):
        VQX=self.VQX
        VXX=self.VXX
        VQQ=self.VQQ
        self.p = shape(VXX)[0]
        alpha = matrix(repeat(1,nQTL))
        a = alpha*VQX*VXX.I*VQX.T*alpha.T
        b = alpha*VQQ*alpha.T
        lamb = trace(VXX)*1.0
        Ci = (1.0*n*VXX + identity(self.p)*lamb).I
        df = trace(Ci*VXX*n*1.0)
        c  = (1.0*df/(n-df-1))*(1.0 - a/b + ve)[0.0]
        r2 = (a/b)**2/(a/b+c)
        r2q = 1 - df*ve/(n+df*ve)
        return [n,df,r2[0,0],r2q]
    def df(self,n,QTL,ve):
        VQX=self.VQX
        VXX=self.VXX
        VQQ=self.VQQ
        self.p = shape(VXX)[0]
        alpha = matrix(repeat(1,nQTL))
        a = alpha*VQX*VXX.I*VQX.T*alpha.T
        b = alpha*VQQ*alpha.T
        lamb = 0.5*self.p*ve
        Ci = (1.0*n*VXX + identity(self.p)*lamb).I
        df = trace(Ci*VXX*n*1.0)
        c  = (1.0*df/(n-df-1))*(1.0 - a/b + ve)[0.0]
        r2 = (a/b)**2/(a/b+c)
        return [n,df,r2[0,0]]

Covarince matrix by Sved's formula¶

In [7]: 
def calcV(L,Ne,p,alleleFreq):
    x = np.arange(0.0,p,1.0)
    V = np.mat(np.zeros((p,p)))
    for i in x:
        for j in x:
            cij = abs(i-j)/p * L
            rij = 0.5*(1 - exp(-2*cij))
            V[i,j] = alleleFreq[i]*alleleFreq[j]*sqrt(1/(4.0*Ne*rij + 1))
    return asmatrix(V)
